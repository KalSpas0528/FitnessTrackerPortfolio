Key Milestones

  • September: Focused on creating the overall design and layout of the website. Mapped out key features and visual elements like the navigation bar and user login system.
  • October: Transitioned into backend development, setting up Supabase for user authentication and Node.js with Express for handling server-side functionality.
  • November - December: Added impactful features like the nutrition tracker and an AI chatbot to enhance user experience.
