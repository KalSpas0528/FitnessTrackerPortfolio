About PowerTitan

PowerTitan is an interactive fitness tracker app designed to help users log their workouts efficiently. By signing up or logging in, users can input data such as exercises, sets, reps, and weights, which is saved and readily accessible for future reference. The app utilizes modern technologies like Supabase, Render, Express, Node.js, and GitHub to store and manage user data, providing a seamless user experience. Whether you're a beginner or a seasoned athlete, PowerTitan helps you track your fitness journey and see measurable progress over time.

I am really into the gym and fitness, and I realized that all the fitness and gym trackers online cost money, so as an aspiring innovator, I took matters into my own hands and decided to create my own.

Key Features

  • User login and registration
  • Logging exercises, sets, reps, and weights
  • Data storage and retrieval using Supabase
  • Track progress with a clean, user-friendly interface
  • Built with modern web technologies (Express, Node.js, GitHub)

Technologies Used

The application is built using the following technologies:

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js, Express, Supabase
  • Hosting: Render
  • Version Control: GitHub

My ultimate goal is to offer users a comprehensive, all-in-one fitness platform that helps them achieve their health goals efficiently and effectively.
