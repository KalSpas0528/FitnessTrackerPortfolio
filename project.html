About PowerTitan

PowerTitan is an all-in-one fitness tracker app designed to empower users in their fitness journey. With this app, users can log their workouts, track nutritional intake, and stay motivated with unique features like a motivational quote generator and an interactive chatbot named Titan AI.

As someone deeply passionate about fitness, I realized most fitness tracking apps charge users. I decided to create PowerTitan as an innovative, free solution for fitness enthusiasts to log and monitor their progress conveniently.

Key Features

  • User Login and Registration: Securely log in or sign up to save and access your fitness data anytime.
  • Workout Tracker: Log exercises, sets, reps, and weights for detailed workout tracking.
  • Nutrition Tracker: Track your meals, calories, and macronutrients like fats, carbs, and protein.
  • Motivational Quote Generator: Get random motivational quotes to inspire your next workout.
  • Interactive Chatbot: Titan AI provides tailored fitness advice and answers your questions.

Technologies Used

The application leverages a robust tech stack to provide a smooth user experience:

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js, Express, Supabase
  • Hosting: Render
  • Version Control: GitHub

How It Works

Once users create an account and log in, they can navigate to the sidebar to access different features:

  • The Add-Workouts page allows users to log exercise data, including sets, reps, and weights, which are saved for future reference.
  • The Nutrition Tracker page helps users log meals and macronutrient information, which is stored and displayed for easy tracking.
  • The Titan AI chatbot offers personalized fitness advice and recommendations based on user queries.

Motivation

PowerTitan reflects my dedication to fitness and technology. By combining these passions, I aimed to create an accessible and free fitness solution for everyone, removing financial barriers to tracking fitness progress effectively.
